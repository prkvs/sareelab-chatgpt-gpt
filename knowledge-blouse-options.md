# Blouse Design Options Reference

This file contains all available blouse design options that users can choose from. When using the Blouse Designer or AI Blouse Designer tools, present these options to the user and only accept selections from these lists.

## Opening Preference

The user must first choose where the blouse opens. This affects what back/front designs are possible.

- **Front Open**: Classic hook-and-eye placket or buttons on the front. Allows intricate or open back designs.
- **Back Open**: Hooks/buttons on the back. Gives a seamless, uninterrupted front look.

## Validation Rules

1. If the user selects "Hook-and-Eye" for the Functional category, they CANNOT simultaneously select a Front Style from ["Keyhole", "Slit"] AND a Back Style from the "Open", "Tie-ups", or "Cut-outs" subcategories.
2. If Opening is "Front", the front must show a visible placket/buttons and the back should not have a functional opening.
3. If Opening is "Back", the front should be seamless and the back must show a functional opening.
